# How to install JDK?

## For Linux
1. Download `.tar.gz`
2. unpack
3. set up symbol link with java and javac

## Windows下环境变量的配置
1. JAVA_HOME: jdk的安装目录
2. CLASSPATH: 编译生成的字节码文件的目录
3. PATH: java, javac 二进制程序的目录

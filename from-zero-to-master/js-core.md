# JavaScript 手记



## 1. 什么是JS？

修改网页的样式

**编写 JS的流程**

1. 布局： HTML+CSS
2. 属性：确定要修改哪些属性
3. 事件：确定用户做哪些操作（产品设计）
4. 编写JS：在事件中，用JS来修改页面元素的样式
